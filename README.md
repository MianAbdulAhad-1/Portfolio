# 🎬 Mian Abdul Ahad — Cinematic Developer Portfolio

> A premium cinematic developer portfolio showcasing immersive web experiences through smooth animations, horizontal storytelling, interactive physics, and modern UI design.

Built with **HTML**, **CSS**, **JavaScript**, **GSAP**, and **Matter.js**, this portfolio transforms a traditional personal website into an engaging digital experience where every section tells a story.

---

## ✨ Overview

This portfolio was created to showcase my skills, projects, and passion for building interactive web experiences as a **Software Engineer**, **Frontend Developer**, and **Creative Developer**.

Rather than presenting information through traditional web pages, the website takes visitors on a **horizontal cinematic journey**, where each section acts as a new scene. Carefully crafted animations, physics-based interactions, and modern UI design work together to create an immersive and memorable experience.

Inspired by award-winning developer portfolios, the project demonstrates my approach to combining **creativity**, **technology**, and **storytelling** into a single seamless experience.

---
